# TechBlog — Explore the World of Tech 🚀

A modern, highly accessible, and content-first technical blogging platform template designed for frontend developers. This project features a clean sidebar layout, a multi-column masonry-style grid for latest articles, custom code-block animations, and complete interactive components like custom mobile navigation and scroll-reveal effects.

## ✨ Features

* **Content-First Responsive Architecture:** Engineered using fluid layouts that bridge layout boundaries gracefully from mobile viewports up to large desktop monitors.
* **Accessible & Semantic Markup (A11y):** Implements strong screen-reader friendliness using explicit ARIA attributes (`aria-expanded`, `aria-label`, `role="navigation"`), semantic native elements, and precise focus management (`tabindex`).
* **Interactive Frontend Elements:** Pre-configured interactive points including a dynamic mobile hamburger menu toggle, article sorting topics, and a smooth "Back to Top" behavior.
* **Modern Code Aesthetic:** Features custom animated presentation containers imitating IDE dark-mode windows with native syntax highlighting styles.
* **Sidebar Layout Integration:** Uses a robust multi-segment presentation balancing main columns with clean widgets (Trending, Topic filters, and Newsletter call-to-actions).

---

## 🛠️ Tech Stack & Concepts Used

* **HTML5:** Semantic architecture, accessible structural layout tags.
* **CSS3:** Flexbox modules, multi-column CSS Grid configurations, custom transitions, and theme design properties.
* **Vanilla JavaScript:** Micro-interactions for toggles, scroll-state handlers, and user experience components.
* **Google Fonts:** Premium structural typography configurations using *Playfair Display*, *DM Sans*, and *JetBrains Mono*.

🚀 How to Run Locally
Follow these simple steps to explore the webpage build on your system:

Download or Clone this project repository directory to your local device.

Ensure that index.html, style.css, and script.js are stored securely inside the same workspace folder.

Launch index.html simply by double-clicking it to spin it up in your active browser.

(Recommended) Run via the Visual Studio Code Live Server extension to easily inspect component dynamics instantly.
