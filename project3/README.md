# Pulse — Interactive Dashboard ⚡

A highly functional, fully accessible, and completely interactive multi-widget control center template. Built to demonstrate advanced **Vanilla JavaScript DOM Manipulation**, this dashboard features live metrics tracking, an advanced task manager, real-time typography metrics analysis, an interactive color engine, and smooth adaptive visual utilities.

## ✨ Features

* **Dynamic Theme Engine (Light/Dark Mode):** Built using an accessible, custom data-attribute switch (`data-theme="light/dark"`) toggled completely via JavaScript with custom hardware-tracked UI transitions.
* **Stateful Task Manager (Tab Panels):** A micro todo-application featuring categorizations (*All*, *Active*, *Done*), empty states handling, live task counter updates, and full clean-up states managed via focusable tab lists.
* **Live Analytics & Numeric Counters:** Native JavaScript asynchronous calculation loops that animate numerical tracking data dynamically on window initialization or manual trigger refreshes.
* **Real-time Input Metrology:** A premium textarea typing monitor updating string length, precise word metrics, reading time predictions, and a custom SVG progress ring loader.
* **RGB Color Mixer Widget:** Dynamic control range input trackers reading values directly, computing outputs instantly into HEX formatting strings, paired with a programmatic click-to-copy utility.
* **Collapsible Accordions (FAQ System):** Engineered entirely for web accessibility standards (using `aria-expanded`, `aria-controls`, and `hidden` properties) complete with a global action expand/collapse override.

---

## 🛠️ Tech Stack & Concepts Used

* **HTML5 Architecture:** Fully semantic structuring combined with modern screen-reader support features (ARIA specification roles).
* **CSS3 Custom Variables:** Managed globally via tokens enabling rapid real-time multi-theme UI shifts and complex presentation scales.
* **Advanced Vanilla JavaScript:** High-efficiency DOM queries, event delegation handling, event string metrics, and direct computational color mutations.
* **Google Fonts Configuration:** High-fidelity typographic branding using *Syne* for futuristic display treatments and *Inter* for ultra-readable interfaces.

🚀 How to Run Locally
Follow these quick actions to view the dashboard interactively on your setup:

Download or Clone this project workspace files locally onto your machine.

Confirm that index.html, style.css, and script.js are positioned together neatly in the same workspace folder.

Launch index.html simply by double-clicking it to spin it up inside your web browser.

(Highly Recommended) Launch utilizing the Visual Studio Code Live Server plugin to test dynamic micro-interactions smoothly.
