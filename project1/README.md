# Zephyr Coffee — Specialty Roasters ☕

A beautiful, fully responsive, and semantic static webpage designed for **Zephyr Coffee**, a fictional specialty coffee roasting brand based in Portland. This project showcases premium typography, modern CSS layouts (Grid & Flexbox), component-driven styling using the BEM methodology, and fluid responsiveness.

## ✨ Features

* **Semantic HTML5:** Built using modern accessible tags like `<header>`, `<main>`, `<section>`, `<article>`, and `<address>`.
* **BEM Methodology:** Highly organized and maintainable CSS classes following the `block__component--modifier` naming convention.
* **Design Tokens (CSS Variables):** Global typography, smooth colors (Espresso, Cream, Rust), spacing scales, and transitions managed centrally.
* **Fully Responsive:** Fluid layouts designed with a mobile-first mindset using modern CSS Media Queries (tested perfectly on Desktop, Tablet, and Mobile devices).
* **Polished Micro-interactions:** Clean hover states, interactive buttons with smooth scale effects, and animated navigation links.

---

## 🛠️ Tech Stack & Concepts Used

* **HTML5** (Structuring & Semantic markup)
* **CSS3** (Flexbox, CSS Grid, Custom Properties, Media Queries)
* **Google Fonts** (*Playfair Display* for classic headers, *DM Sans* for sleek body text)
* **BEM Architecture** (Block, Element, Modifier)

🚀 How to Run Locally
Follow these simple steps to view the project on your local machine:

Clone or Download this repository folder.

Make sure both index.html and style.css are in the same directory.

Double-click index.html to open it instantly in your default web browser.

(Alternative) Right-click index.html and use the Live Server extension in VS Code for live changes.
